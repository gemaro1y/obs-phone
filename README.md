# PhoneStream

Телефон как камера для OBS Studio. Работает через браузер — никаких приложений.

## Как это работает

```
Телефон (браузер) → canvas → JPEG кадры → Socket.IO → Сервер → MJPEG поток → OBS
```

Телефон захватывает видео через `getUserMedia` или `getDisplayMedia`, кодирует кадры в JPEG через canvas и отправляет на сервер. OBS подключается как Browser Source и видит MJPEG-поток в реальном времени.

## Возможности

- **Камера или демонстрация экрана** — выбирайте что стримить
- **720p / 1080p** — настройка разрешения
- **30 / 60 FPS** — выбор частоты кадров
- **Передняя / задняя камера** — переключение одной кнопкой
- **Альбомная / книжная ориентация** — авто или ручной выбор
- **Микрофон** — с индикатором уровня сигнала и кнопкой мута
- **QR-код** — быстрое подключение телефона
- **Статистика** — разрешение, FPS в реальном времени

## Установка

```bash
git clone https://github.com/gemaro1y/obs-phone.git
cd obs-phone
npm install
```

## Запуск

```bash
npm run server
```

## Настройка

### OBS Studio

1. Откройте OBS → Источники → `+` → **Браузер**
2. URL: `http://localhost:4801/stream`
3. Ширина: `1920`, Высота: `1080`
4. Уберите галочку «Локальные файлы»
5. Нажмите «ОК»

### Телефон

1. Откройте `https://<IP_ПК>:4800/mobile` в браузере телефона
2. При предупреждении о сертификате нажмите «Расширенные» → «Перейти»
3. Разрешите доступ к камере и микрофону
4. Выберите настройки (источник, разрешение, камера)
5. Нажмите «Трансляция»

## Порты

| Порт  | Протокол | Назначение |
|-------|----------|------------|
| 4800  | HTTPS    | Мобильная страница (камера требует HTTPS) |
| 4801  | HTTP     | OBS Browser Source + MJPEG-поток |

## Структура проекта

```
phone-stream/
├── server.js          — Express + Socket.IO + MJPEG сервер
├── start.js           — Запуск без Electron
├── main.js            — Electron main process
├── preload.js         — Electron preload
├── mobile/            — Мобильная PWA
│   ├── index.html     — UI с настройками
│   ├── style.css      — Dark theme, Lucide иконки
│   └── app.js         — Захват камеры/экрана, JPEG-кодирование
├── stream/            — OBS Browser Source
│   └── index.html     — MJPEG-плеер
├── renderer/          — Electron dashboard
│   ├── index.html     — QR-код, статистика
│   └── style.css      — Desktop UI
└── cert/              — Самоподписанный сертификат (генерируется автоматически)
```

## Технологии

- **Node.js** — сервер
- **Express** — HTTP/HTTPS
- **Socket.IO** — сигналинг и передача кадров
- **node-forge** — генерация SSL-сертификата
- **Lucide** — иконки UI
- **Canvas API** — JPEG-кодирование на телефоне

## Удалённый доступ (Tailscale)

Для стрима из любой точки мира:

1. Установи Tailscale на ПК и телефон
2. Войди одним аккаунтом на обоих
3. Сервер автоматически покажет Tailscale IP (`100.x.x.x`)
4. На телефоне открой `https://<Tailscale_IP>:4800/mobile`
